\subsection{Descrição Detalhada}
Interface com funcoes de utilidade para a J\+VM operar. 

Interface responsavel por implementar os servicos uteis para a J\+VM, com submodulos responsaveis por\+:
\begin{DoxyItemize}
\item Decoder\+: Decodificacao de metodos, campos e estuturas que simulam o acesso a bibliotecas Java.
\item Conversion\+: Servicos relacionados a conversao de tipos.
\item java.\+io\+: Servicos que simulam os metodos e atributos da biblioteca java.\+io
\item java.\+lang\+: Servicos que simulam os metodos e atributos da biblioteca java.\+lang 
\end{DoxyItemize}
